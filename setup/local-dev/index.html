Local Development Setup

Prerequisites

  • Node.js vXX+
  • Docker (optional)
  • MongoDB/Postgres locally or remote

Steps

  1. Clone repo: git clone <repo-url>
  2. Install dependencies: npm install
  3. Setup .env file (copy .env.example)
  4. Run local server: npm run dev
